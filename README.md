# Custom distortion pedal implemented in kicad and ltspice

## This pedal is comprised of 3 stages:
  - Amplification
  - Clipping
  - Output Filtering

## The main branch contains a 100mm X 100mm PCB, while the 1590b-pcb branch contains a 35mm X 88mm PCB, which fits inside a 1590b box
